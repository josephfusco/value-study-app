# Value Study

A browser-based tool for converting images to posterized grayscale. Useful for analyzing tonal values in reference images.

## Features

- Drag-and-drop or click to upload images
- Configurable value levels (3, 5, or 9)
- Contrast adjustment (-100 to +100)
- Export as PNG

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Radix UI

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
